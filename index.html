Recordando:

  • Javascript en HTML(script)

    - Document.querySelector(): Devuelve el primer elemento del documento que coincida con el grupo especificado de selectores. En este caso esta función actúa de forma muy similar a un getElementById().

    - addEventListener(): Registra un evento a un objeto en específico. Su sintaxis es muy sencilla: target.addEventListener('tipo de evento',función_a_lanzar,booleano).

  • Google font icons

    - Icono hamburguer: 1. Copiamos el link del css y lo pegamos en el html, 2. Copiamos el estilo css y lo pegamos en nuestra hoja css, y 3. Copiamos el codigo span(codigo con clase y texto) y lo pegamos dentro de nuestro boton.

Tarjetas animadas

Recordando:

  • Box-shadow

    - Propiedades de box-shadow: 1. h-offset(desplazamiento horizontal de la sombra), 2. v-offset(desplazamiento vertical de la sombra), 3. blur(desenfoque, cuanto mayor sea el numero, mas borrosa será la somnra), 4. spread(propagación, valor positivo aumenta el tamaño de la sombra, valor negativo disminuye), color, inset (cambia la sombra, de una sombra exterior a una sombra interior).

  • Transition

    - Propiedades de Transition: 1. transition-property(Especifica el nombre o nombres de las propiedades CSS a las que deberían aplicarse las transiciones), 2. transition-duration(Especifica la duración en la que sucederán las transiciones), 3. transition-timing-function(Especifica la curva de velocidad del efecto de transición), 4. transition-delay(especifica un retraso (en segundos) para el efecto de transición)
